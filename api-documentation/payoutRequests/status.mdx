---
title: Status
---

| Status       | Description                                                                                                                |
| ------------ | -------------------------------------------------------------------------------------------------------------------------- |
| `processing` | The request has been sent to the bank and Palomma is waiting for response from the bank.                                   |
| `approved`   | The payout has been successful.                                                                                            |
| `cancelled`  | Error state. More detail available in `statusMessage` field. See all [Status Messages](/api-documentation/statusMessages). |
